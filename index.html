I created this database for a project when I could not find a proper API to use when creating a React Menu. I hope it serves you well!

Getting Started

There are no restrictions or keys. Feel free to use this database as you choose!

This is hosted on a heroku server. For those unfirmiliar with Heroku, the servers are free to use up until a certain threshold. However, the servers go to "sleep" after about 30min. Your first attempt at accessing the database will take the longest. After this the server should be "awake" and your access should be almost instant.
